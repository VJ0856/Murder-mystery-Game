# 🎮 Murder Mystery - Console vs Web Comparison

## Quick Summary

### Console Version (`main.py`)
- **Best For:** Quick gameplay, minimal setup
- **Interface:** Text-based, terminal
- **Launch:** `python main.py`
- **Setup Time:** Instant
- **Learning Curve:** Minimal

### Web Version (`app.py`)
- **Best For:** Visual experience, professional look
- **Interface:** Beautiful GUI with detective theme
- **Launch:** `streamlit run app.py`
- **Setup Time:** Loads in seconds
- **Learning Curve:** Very easy

## Installation Comparison

### Console Setup
```bash
pip install -r requirements.txt
python main.py
```

### Web Setup
```bash
pip install -r requirements.txt
streamlit run app.py
```

**Both are equally simple!**

## When to Use Each Version

### Use Console When:
- You're in a terminal/SSH environment
- You want the fastest possible startup
- You're on a system without browser access
- You prefer text-based interfaces
- You want to run it headless
- You like command-line tools

### Use Web When:
- You're on a desktop/laptop
- You have a modern web browser
- You want a beautiful experience
- You want to share with others
- You like visual feedback
- You're using a touch device
- You want professional presentation
- You're not familiar with terminals
