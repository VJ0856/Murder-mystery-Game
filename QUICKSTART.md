# 🕵️ Murder Mystery Game - Quick Start

## ✨ **Super Easy Setup**

Just 2 steps and you're ready to play!

### Step 1: Install Required Package
Open PowerShell or Command Prompt and run:
```bash
pip install -r requirements.txt
```

### Step 2: Play the Game!
```bash
python main.py
```

That's it! The API key is already set up. No more configuration needed! 🎉

---

## 🎮 **How to Play**

The game is designed to be beginner-friendly:

1. **Read the welcome** - You'll get tips on how to play
2. **View the crime scene** - Understand where the crime happened
3. **Review clues** - Look at all the evidence collected
4. **Interrogate suspects** - Question each suspect about their alibi
5. **Find contradictions** - Match clues against alibis to catch lies
6. **Make your accusation** - Accuse the killer before time runs out!

### 💡 Tips:
- You have **5 interrogations maximum** - use them wisely!
- Each contradiction you find = **10 points**
- Correct final accusation = **25 bonus points**
- Match clues to alibis to find contradictions
- Pay attention to what seems odd or inconsistent
