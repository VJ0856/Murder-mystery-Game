# 🌐 Quick Start - Web Version

## 30-Second Setup

### Step 1: Install (2 minutes)
```bash
pip install -r requirements.txt
```

### Step 2: Run (1 command)
```bash
streamlit run app.py
```

### Step 3: Play!
- Browser opens automatically
- Click "START NEW INVESTIGATION"
- Play the game!

**That's it!** 🎉

---

## On Windows? Even Easier!

Just double-click:
```
run_web.bat
```

Done! Browser opens and game starts.

## Game Controls

| Action | How |
|--------|-----|
| **Start Game** | Click "START NEW INVESTIGATION" |
| **View Crime Scene** | Click "Crime Scene" tab |
| **View Suspects** | Click "Suspects" tab |
| **Interrogate** | Click suspect's "Interrogate" button |
| **View Clues** | Click "Clues" tab |
| **Make Accusation** | Click "Accusation" tab → Click suspect |
| **New Game** | Click "NEW INVESTIGATION" button |

---

## What to Do First

1. **Click "START NEW INVESTIGATION"**
   - Wait 2-3 seconds for AI to generate case
   - See "Mystery case generated successfully!"

2. **Check "Crime Scene" Tab**
   - Read about the victim
   - Learn location and evidence

3. **Check "Clues" Tab**
   - Read all 8 clues
   - Think about what they mean

4. **Go to "Suspects" Tab**
   - See all 3 suspects
   - Click interrogate buttons

5. **When ready**
   - Go to "Accusation" tab
   - Choose who the murderer is

## Troubleshooting

**"Command not found: streamlit"**
```bash
pip install streamlit
```

**"The app won't open in browser"**
- Check terminal output
- Manually go to: `http://localhost:8501`

**"AI is taking too long"**
- Normal! It's generating your unique case
- Takes 2-3 seconds max

**"My score didn't update"**
- Refresh the tab: F5
- Click another tab and come back
