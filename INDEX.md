## 🎯 Game Overview

**Objective:** Find the murderer before time runs out!

**Game Mechanics:**
- 3 suspects to interrogate
- 5 maximum interrogations
- 8 clues to analyze
- Find contradictions = points
- Make final accusation

**Scoring:**
- +10 points per contradiction found
- +25 bonus for correct accusation
- Every game is unique!
