# 🕵️ Murder Mystery Investigation Game

A dynamic, AI-powered murder mystery game available in both console and web versions!

## 🎮 Two Versions Available

### Console Version
- Classic text-based interface
- Run directly in terminal
- Quick and simple
- Run with: `python main.py`

### Web Version (Streamlit)
- Beautiful detective-themed UI
- Interactive web interface
- Sidebar dashboard
- Suspect cards and clue display
- Run with: `streamlit run app.py`

## 🚀 Quick Start

### Option 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Option 2: Console Version
```bash
python main.py
```

### Option 3: Web Version
```bash
streamlit run app.py
```

Or double-click: `run_web.bat` (Windows)

## ✨ Features

### AI-Powered Content Generation
- Gemini AI generates unique cases every game
- New victim, suspects, and clues each time
- Logical contradictions between clues and alibis
- Fresh storylines guaranteed

### Game Mechanics
- 3 suspects to interrogate
- 5 maximum interrogations per game
- 8 unique clues to review
- Find contradictions for bonus points
- Make your accusation to solve the case

### Scoring System
- +10 points per contradiction found
- +25 bonus points for correct accusation
- Track your detective score
- Try to get the highest score

## 🎯 How to Play

1. **Start a Case** - Begin a new investigation
2. **Review Evidence** - Check crime scene and clues
3. **Interrogate Suspects** - Question suspects about alibis
4. **Find Contradictions** - Match clues against statements
5. **Make Accusation** - Accuse the killer before time runs out
6. **Check Results** - See if you solved it correctly

## 🕵️ Investigation Tips

- Listen carefully to alibis
- Match clues against what suspects claim
- Contradictions are your evidence
- Use interrogations strategically
- Think before making your accusation
- Look for inconsistencies in stories
